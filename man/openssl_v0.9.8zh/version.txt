       version - print OpenSSL version information

       ooppeennssssll vveerrssiioonn [--aa] [--vv] [--bb] [--oo] [--ff] [--pp]

       --vv  the current OpenSSL version.

       --bb  the date the current version of OpenSSL was built.

       --oo  option information: various options set when the library was built.

       --cc  compilation flags.

       --pp  platform setting.

       --dd  OPENSSLDIR setting.

       The output of ooppeennssssll vveerrssiioonn --aa would typically be used when sending
       in a bug report.
